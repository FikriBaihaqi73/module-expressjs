## 1. Apa itu Database? (SQL vs NoSQL)

Bayangkan database sebagai **gudang penyimpanan data** aplikasi kita.

### SQL (Relational Database)
Data disimpan dalam **tabel** (baris & kolom) seperti Excel, tapi jauh lebih canggih.
- **Terstruktur (Schema Strict):** Kita harus menentukan kolom dan tipe datanya di awal.
- **Relasional:** Tabel-tabel bisa saling terhubung (misal: Tabel User terhubung ke Tabel Order).
- **Contoh:** PostgreSQL, MySQL.
- **Cocok untuk:** E-commerce, Keuangan, Sistem Inventori (data yang butuh konsistensi tinggi).

### NoSQL (Non-Relational)
Data disimpan secara fleksibel (dokumen JSON, key-value).
- **Fleksibel:** Tidak butuh skema ketat di awal.
- **Contoh:** MongoDB, Redis.
- **Cocok untuk:** Data log, social media feed, data yang strukturnya sering berubah.

**Kenapa PostgreSQL?**
PostgreSQL adalah "Raja"-nya database open-source saat ini. Stabil, kaya fitur, dan menjadi standar industri untuk backend modern (banyak dipakai startup hingga unicorn).

## 2. Instalasi PostgreSQL

### Windows
1. Download installer dari [situs resmi PostgreSQL](https://www.postgresql.org/download/windows/).
2. Jalankan installer.
3. Saat diminta password untuk user `postgres`, **ingat baik-baik passwordnya!** (Misal: `root` atau `admin`).
4. Stack Builder boleh di-skip (uncheck) setelah instalasi selesai.
5. Buka aplikasi **pgAdmin** atau terminal **SQL Shell (psql)** yang ikut terinstall.

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
```
Secara default, Linux membuat user sistem bernama `postgres`. Masuk dengan:
```bash
sudo -i -u postgres
psql
```

### Mac OS
Gunakan [Postgres.app](https://postgresapp.com/) (paling mudah) atau Homebrew:
```bash
brew install postgresql
brew services start postgresql
```

### Cek Instalasi
Buka terminal, ketik:
```bash
psql --version
```

### Masuk ke Console PostgreSQL
```bash
psql postgres
```
(Keluar dengan ketik `\q`)

### Rekomendasi Tools GUI
Selain terminal, sangat disarankan menginstall aplikasi GUI agar lebih mudah melihat data:
- **DBeaver** (Gratis, Universal) - *Recommended*
- **TablePlus** (Ringan, UI Bagus)
- **pgAdmin** (Bawaan PostgreSQL)

## 3. Basic SQL Query (Bahasa Database)

Kita akan menggunakan bahasa **SQL (Structured Query Language)** untuk menyuruh database melakukan sesuatu.

Kita akan menggunakan bahasa **SQL (Structured Query Language)** untuk menyuruh database melakukan sesuatu.

### A. Tipe Data Umum di PostgreSQL
Sebelum membuat tabel, kenalan dulu dengan tipe data yang sering dipakai:
1. **INT / INTEGER**: Angka bulat (contoh: stok, umur).
2. **VARCHAR(n)**: Teks dengan batas panjang `n` karakter (contoh: nama, email).
3. **TEXT**: Teks panjang tanpa batas (contoh: deskripsi, artikel).
4. **BOOLEAN**: Benar/Salah (`true`/`false`).
5. **DECIMAL(p,s)**: Angka desimal/koma, cocok untuk uang (contoh: harga).
6. **TIMESTAMP**: Tanggal dan waktu lengkap.

### B. Membuat Database
```sql
CREATE DATABASE ecommerce_db;
```
Masuk ke database tersebut:
```bash
\c ecommerce_db
```

### C. Membuat Tabel (CREATE TABLE)
Kita buat tabel `products` untuk menyimpan barang jualan.
```sql
CREATE TABLE products (
    id SERIAL PRIMARY KEY,          -- ID otomatis nambah (1, 2, 3...)
    name VARCHAR(100) NOT NULL,     -- Nama produk, max 100 huruf, wajib isi
    description TEXT,               -- Deskripsi bebas panjangnya
    price DECIMAL(10, 2) NOT NULL,  -- Harga, 10 digit, 2 desimal
    stock INT DEFAULT 0,            -- Stok, default 0 kalau gak diisi
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP -- Waktu dibuat otomatis
);
```

### D. Menambah Data (INSERT)
```sql
INSERT INTO products (name, description, price, stock) 
VALUES ('Laptop Gaming', 'Intel i7, RTX 3060', 15000000, 10);

INSERT INTO products (name, price, stock) 
VALUES ('Mouse Wireless', 250000, 50);
```

### E. Mengambil Data (SELECT)
```sql
-- Ambil semua kolom
SELECT * FROM products;

-- Ambil nama dan harga saja
SELECT name, price FROM products;

-- Ambil dengan filter (WHERE)
-- Ambil dengan filter (WHERE)
SELECT * FROM products WHERE price > 1000000;

-- Operator Filter Lainnya:
-- =  (Sama dengan)
-- != (Tidak sama dengan)
-- >  (Lebih besar)
-- <  (Lebih kecil)
-- LIKE (Mencari pola teks)
SELECT * FROM products WHERE name LIKE '%Laptop%'; -- Cari yang namanya mengandung kata 'Laptop'

-- Mengurutkan Data (ORDER BY)
SELECT * FROM products ORDER BY price DESC; -- Termahal ke termurah (ASC untuk termurah ke termahal)

-- Membatasi Jumlah Data (LIMIT)
SELECT * FROM products LIMIT 5; -- Cuma ambil 5 data teratas
```

### F. Update Data (UPDATE)
**HATI-HATI:** Selalu pakai `WHERE`, kalau tidak semua data berubah!
```sql
UPDATE products 
SET price = 14500000, stock = 8 
WHERE id = 1;
```

### G. Hapus Data (DELETE)
**HATI-HATI:** Selalu pakai `WHERE`!
```sql
DELETE FROM products WHERE id = 2;
```
