## 1. Apa itu Database? (SQL vs NoSQL)

Bayangkan database sebagai **gudang penyimpanan data** aplikasi kita.
- **SQL (Relational)**: Data disimpan dalam **tabel** (baris & kolom). Terstruktur, ketat, dan punya relasi.
  - Contoh: PostgreSQL, MySQL.
  - Cocok untuk: E-commerce, Keuangan, Sistem yang butuh konsistensi tinggi.
- **NoSQL (Non-Relational)**: Data disimpan fleksibel (dokumen, key-value).
  - Contoh: MongoDB, Redis.
  - Cocok untuk: Data yang strukturnya berubah-ubah, Big Data.

**Kenapa PostgreSQL?**
PostgreSQL adalah database open-source tercanggih saat ini. Stabil, kaya fitur, dan standar industri untuk backend modern.

## 2. Instalasi PostgreSQL

### Windows
1. Download installer dari [situs resmi PostgreSQL](https://www.postgresql.org/download/windows/).
2. Jalankan installer.
3. Saat diminta password untuk user `postgres`, **ingat baik-baik passwordnya!** (Misal: `root` atau `admin`).
4. Stack Builder boleh di-skip (uncheck) setelah instalasi selesai.
5. Buka aplikasi **pgAdmin** atau terminal **SQL Shell (psql)** yang ikut terinstall.

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
```
Secara default, Linux membuat user sistem bernama `postgres`. Masuk dengan:
```bash
sudo -i -u postgres
psql
```

### Mac OS
Gunakan [Postgres.app](https://postgresapp.com/) (paling mudah) atau Homebrew:
```bash
brew install postgresql
brew services start postgresql
```

### Cek Instalasi
Buka terminal, ketik:
```bash
psql --version
```

### Masuk ke Console PostgreSQL
```bash
psql postgres
```
(Keluar dengan ketik `\q`)

## 3. Basic SQL Query (Bahasa Database)

Kita akan menggunakan bahasa **SQL (Structured Query Language)** untuk menyuruh database melakukan sesuatu.

### A. Membuat Database
```sql
CREATE DATABASE ecommerce_db;
```
Masuk ke database tersebut:
```bash
\c ecommerce_db
```

### B. Membuat Tabel (CREATE TABLE)
Kita buat tabel `products` untuk menyimpan barang jualan.
```sql
CREATE TABLE products (
    id SERIAL PRIMARY KEY,          -- ID otomatis nambah (1, 2, 3...)
    name VARCHAR(100) NOT NULL,     -- Nama produk, max 100 huruf, wajib isi
    description TEXT,               -- Deskripsi bebas panjangnya
    price DECIMAL(10, 2) NOT NULL,  -- Harga, 10 digit, 2 desimal
    stock INT DEFAULT 0,            -- Stok, default 0 kalau gak diisi
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP -- Waktu dibuat otomatis
);
```

### C. Menambah Data (INSERT)
```sql
INSERT INTO products (name, description, price, stock) 
VALUES ('Laptop Gaming', 'Intel i7, RTX 3060', 15000000, 10);

INSERT INTO products (name, price, stock) 
VALUES ('Mouse Wireless', 250000, 50);
```

### D. Mengambil Data (SELECT)
```sql
-- Ambil semua kolom
SELECT * FROM products;

-- Ambil nama dan harga saja
SELECT name, price FROM products;

-- Ambil dengan filter (WHERE)
SELECT * FROM products WHERE price > 1000000;
```

### E. Update Data (UPDATE)
**HATI-HATI:** Selalu pakai `WHERE`, kalau tidak semua data berubah!
```sql
UPDATE products 
SET price = 14500000, stock = 8 
WHERE id = 1;
```

### F. Hapus Data (DELETE)
**HATI-HATI:** Selalu pakai `WHERE`!
```sql
DELETE FROM products WHERE id = 2;
```
