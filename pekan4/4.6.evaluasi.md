## SPESIFIKASI FINAL PROJECT

Kamu diminta membangun **Backend API untuk Marketplace** (mirip Tokopedia/Shopee versi lite).

### 1. Fitur Utama (Wajib)
1.  **Auth:** Register, Login, JWT, Role (Admin vs User).
2.  **Product:** CRUD Produk, Upload Gambar, Kategori.
3.  **Cart & Order:**
    - User bisa tambah barang ke keranjang (Opsional, bisa langsung checkout).
    - User Checkout (Transaction N-N).
    - Stok berkurang otomatis.
4.  **History:** User bisa lihat riwayat belanja.

### 2. Persyaratan Teknis
- **Bahasa:** TypeScript.
- **Framework:** Express.js.
- **Database:** PostgreSQL + Prisma ORM.
- **Architecture:** Repository-Service Pattern (Clean Code).
- **Validation:** Input validasi lengkap.
- **Error Handling:** Global Error Handler.
- **Deployment:** Live di Railway/Render.

### 3. Bonus (Nilai Plus)
- Dokumentasi Swagger.
- Unit Testing (Jest).
- Fitur Search & Filter canggih.
